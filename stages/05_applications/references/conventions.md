# Tracker Conventions

- Hiring stages: **Preparing** → **Applied** → **Interview** (pill classes: saved-stage, applied-stage,
  interview-stage).
- Every application record holds: role, company, dates, stage, materials status (résumé version, cover
  letter), contact, exactly one next action, and its due date.
- An application with no next action or no due date is a defect to surface, not a normal state.
